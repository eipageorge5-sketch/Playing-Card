# 🃏 Higher or Lower? - Playing Card Game

A sleek, responsive web-based card game built with **HTML5, CSS3, and Vanilla JavaScript**. Test your intuition by guessing if the next card in the deck will be higher or lower than the current one!

## 🚀 Features
*   **Realistic Deck Logic:** Uses a 52-card deck with a **Fisher-Yates shuffle** algorithm for true randomness.
*   **Interactive UI:** Smooth CSS animations (pop and shake) for instant feedback on guesses.
*   **High Score Tracking:** Saves your best score locally in your browser using `localStorage`.
*   **Responsive Design:** Optimized for both desktop and mobile play with a "Poker Table" aesthetic.

## 🎮 How to Play
1.  **Start:** A random card is drawn from the shuffled deck.
2.  **Guess:** Click **Higher ↑** if you think the next card's value is greater, or **Lower ↓** if you think it's smaller.
    *   *Note: Aces are high (14), Kings are 13, etc.*
3.  **Score:** Get points for every correct guess. If you're wrong, try to beat your High Score in the next round!
4.  **Reset:** Use the **Shuffle & Reset** button at any time to start with a fresh 52-card deck.

## 🛠️ Built With
*   **HTML5** - Semantic structure.
*   **CSS3** - Flexbox layout and Keyframe animations.
*   **JavaScript** - Game logic, DOM manipulation, and Local Storage.
